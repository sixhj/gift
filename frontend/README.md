# gift

A new Flutter project.


## 安装
```bash
flutter pub get
flutter create .
```

## 运行
```bash
# Web
flutter run -d chrome

# Windows
flutter run -d windows
```

## 打包
```bash
# Web
flutter build web

# Windows
flutter build windows

# Android
flutter build apk
```
